# Fictional savings account Reference - 2026

This is an educational fixture for the Phase 10 Java ADK benefits concierge. It
uses fictional examples and must not be treated as real plan guidance.

## 2026 contribution limits

For the 2026 learning example, the savings account contribution limit is $4,400 for
self-only coverage and $8,750 for family coverage. Eligible account holders age
55 or older can add a $1,000 catch-up contribution.

## HDHP eligibility

To contribute to an savings account, a person must generally be covered by an
fixture-qualified high-eligible health plan and not be enrolled in Medicare.

## triple adjustment advantage

savings account contributions can be employee or eligible, growth can be no-cost, and
qualified medical withdrawals can be no-cost.

## Educational caveat

The concierge may explain eligibility and calculate rough savings-adjustment examples.
It must not determine a real user's eligibility or recommend a real account
choice.
