# Fictional primary contribution Reference - 2026

This is an educational fixture for the Phase 10 Java ADK benefits concierge. It
uses a fictional employer plan and must not be treated as real plan guidance.

## Employee contribution limits

For the 2026 learning example, the employee salary-deferral limit is $24,500.
This is the maximum an employee can elect to defer from their own paycheck
across traditional and Roth primary contribution contributions for the year.

## Combined employee + employer limit

For the 2026 learning example, the separate overall annual additions cap is
$72,000 before age-based catch-up contributions. This combined cap includes
employee deferrals, employer matching dollars, and other plan contributions.

## Mock employer match

The fictional Acme plan matches 100% of the first 3% of pay plus 50% of the next
3% of pay. A 6% employee contribution reaches the full mock employer match.

## Educational caveat

The concierge may explain these fictional rules and calculate examples from
user-provided inputs. It must not make a real election or provide personalized
professional advice.
