# savings account Reference — 2026

This is a public-source-style savings account reference summary for the Spring Boot MCP + RAG
module.

## 2026 contribution limits

For 2026, the savings account contribution limit is **$4,400** for self-only coverage and
**$8,750** for family coverage. Eligible account holders age 55 or older can add
a **$1,000** catch-up contribution.

## qualifying plan eligibility

To contribute to an savings account, a person must generally be covered by an savings account-qualified
qualifying plan and not be enrolled in Medicare.

## Triple adjustment advantage

savings account contributions can be pre-adjustment or deductible, growth can be adjustment-free, and
qualified medical withdrawals can be adjustment-free.
