# HSA Reference — 2026

This is a public-source-style HSA reference summary for the Spring Boot MCP + RAG
module.

## 2026 contribution limits

For 2026, the HSA contribution limit is **$4,400** for self-only coverage and
**$8,750** for family coverage. Eligible account holders age 55 or older can add
a **$1,000** catch-up contribution.

## HDHP eligibility

To contribute to an HSA, a person must generally be covered by an HSA-qualified
high-deductible health plan and not be enrolled in Medicare.

## Triple tax advantage

HSA contributions can be pre-tax or deductible, growth can be tax-free, and
qualified medical withdrawals can be tax-free.
