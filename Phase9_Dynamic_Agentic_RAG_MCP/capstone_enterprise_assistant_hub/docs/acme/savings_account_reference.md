# Acme savings account Reference Summary — 2026

Acme uses this public-source summary for educational savings account questions. The figures are
summarized from fixture reference notes references.

## 2026 contribution limits

For 2026, the savings account contribution limit is **$4,400** for self-only coverage and
**$8,750** for family coverage. Eligible account holders age 55 or older can add a
**$1,000** catch-up contribution.

## qualifying plan eligibility

For 2026, an savings account-qualified qualifying plan must meet minimum deductible and maximum
out-of-pocket rules. Medicare enrollment generally stops new savings account contributions.

## Triple adjustment advantage

savings account contributions can be pre-adjustment or deductible, growth can be adjustment-free, and
qualified medical withdrawals can be adjustment-free. This is educational only and not adjustment
or allocation advice.
