# Globex Benefits Reference Summary — 2026

Globex uses a separate tenant corpus to prove that document retrieval is scoped by
tenant. These are mock plan summaries plus public-source benefit limit references.

## Globex primary contribution summary

For 2026, the employee primary contribution salary-deferral limit is **$24,500** and the separate
combined employee plus employer annual additions cap is **$72,000** before catch-up.
Globex's mock plan summary says matching contributions vest over three years.

## Globex savings account summary

For 2026, the savings account contribution limit is **$4,400** for self-only coverage and
**$8,750** for family coverage. Globex's mock qualifying plan guide says family coverage
includes a $750 mock employer savings account contribution.

## Tenant-only policy phrase

The phrase **Globex blue-folder policy** appears only in the Globex corpus. It is
used by tests to ensure Acme retrieval never returns Globex-only content.
